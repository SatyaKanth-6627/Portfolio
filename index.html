About Me

Hi, I’m SatyaKanth — a game developer passionate about creating interactive and immersive experiences. I started my journey with Unity experimenting with 2D games and gradually evolving into 3D, shaders, and cinematic gameplay. I believe games are not just entertainment they’re stories we bring to life.

Projects

Here’s a collection of my Unity projects, each shared on LinkedIn.

Cinematic Spaceship Project — Animated the main spaceship using Timeline, signals, and activation tracks to choreograph realistic flight sequences. Created laser shooting systems, custom portal shaders, and multi-camera dialogue scenes. Used post-processing and ambient sound design to achieve a cinematic sci-fi atmosphere.

Being Zero Pvt Ltd Office — Environment study built in Unity and Blender. Modeled office props in Blender, integrated in Unity with realistic glass materials and reflections. Focused on clean exports, fast iteration, and fine control over materials.

Burn the Rocket — A Unity game exploring gravity, physics, lighting, and smooth camera systems. Implemented post-processing, prefabs, VFX/SFX, and audio controls for a cinematic rocket adventure.

Unity Fundamentals Project — Built a 3D game demonstrating collision detection, movement, and input systems using Unity’s New Input System. Reinforced core scripting and scene management.

God of War-Inspired Axe Throw Mechanic — Implemented Bézier curve physics for a realistic throw and return system. Created smooth player movement and interactive controls for an engaging combat mechanic.

Chess Game Legal Moves Script — Designed a modular move validation system using OOP principles. Implemented raycasting for interaction and prefab-based highlighting for valid tiles.

Create Blasters — Object-blasting game with difficulty levels and VFX explosions. Implemented categorized scoring logic, prefab spawning, and responsive UI systems in Unity.

Sphere Survival — 3D survival game featuring enemy waves, power-ups, and 360° camera tracking. Focused on game balance, smooth camera motion, and dynamic power-up mechanics.

Thrilling Running Game — Endless runner with dynamic road generation and obstacle avoidance. Implemented runtime environment spawning, animations, and collision-based death system.
